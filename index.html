    • Symfony2colors

    A color theme for PHPStorm, as seen in the Symfony2 documentation

    Project maintained by cordoval Hosted on GitHub Pages — Theme by mattgraham

    Based on the official Symfony2 documentation colors palette

    Because we care about style, comfort, and consistency, we have created a theme for the PHPStorm IDE based on the color palette used in the Symfony2 documentation. This theme is a plesent, dark-colored theme, with a great contrast of elements.

    This theme provides a unified color palette for your PHP, Twig, HTML, XML, YAML, JavaScript, and CSS files
